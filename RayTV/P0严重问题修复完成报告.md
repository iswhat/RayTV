## 问题1: ConfigSourceService 缺失方法

### 问题描述
SourcePoolManager 调用了以下不存在的方法：
- `getAllSources()` - 获取所有配置源
- `getParsedConfig(url)` - 从URL获取解析后的配置
- `getLiveDataFromUrl(url)` - 从URL获取直播数据

### 解决方案
将 `ConfigSourceService.ts` 转换为 `ConfigSourceService.ets` 并添加缺失的方法。

#### 1.1 添加 `getAllSources()` 方法
```typescript
public async getAllSources(): Promise<ConfigSource[]> {
  try {
    const sources = await StorageUtil.getItem<ConfigSource[]>(CONFIG_SOURCE_STORAGE_KEY);
    return sources || [];
  } catch (error) {
    this.logger.error('Failed to get all sources', error);
    return [];
  }
}
```

**功能**:
- 获取存储中的所有配置源，不进行过滤
- 返回所有配置源的数组
- 错误时返回空数组

#### 1.2 添加 `getParsedConfig()` 方法
```typescript
public async getParsedConfig(url: string): Promise<ParsedConfig | null> {
  try {
    const result = await this.loadConfigSource(url);
    if (!result.isSuccess() || !result.data) {
      return null;
    }

    const parsedResult = await this.parseConfigContent(JSON.stringify(result.data));
    if (!parsedResult.isSuccess() || !parsedResult.data) {
      return null;
    }

    return parsedResult.data;
  } catch (error) {
    this.logger.error(`Failed to get parsed config from ${url}`, error);
    return null;
  }
}
```

**功能**:
- 从指定URL获取配置内容并解析
- 返回解析后的配置对象
- 失败时返回 null

#### 1.3 添加 `getLiveDataFromUrl()` 方法
```typescript
public async getLiveDataFromUrl(url: string): Promise<LiveSource[]> {
  try {
    const parsedConfig = await this.getParsedConfig(url);
    if (!parsedConfig || !parsedConfig.lives) {
      return [];
    }
    return parsedConfig.lives;
  } catch (error) {
    this.logger.error(`Failed to get live data from ${url}`, error);
    return [];
  }
}
```

**功能**:
- 从指定URL获取直播频道数据
- 返回直播频道数组
- 失败时返回空数组

#### 1.4 文件转换

- 修复 ArkTS 兼容性问题（`any` 类型改为 `unknown` 和 `Record<string, Object>`）
- 删除旧的 `.ts` 文件
- 更新所有导入路径

```typescript
private async loadContentStats(): Promise<void> {
  if (!this.tempSource) return;

  this.loading = true;
  this.showLoadError = false;

      adBlockRules: adBlockRules.length,
      decoderConfigs: decoderConfigs.length,
      parserConfigs: 0
    };

    this.showLoadError = true;
    this.errorMessage = '加载内容统计失败';
    // 失败时使用默认值
    this.contentStats = {
      vodSites: 0,
      liveChannels: 0,
      wallpapers: 0,
      adBlockRules: 0,
      decoderConfigs: 0,
      parserConfigs: 0
    };
  } finally {
    this.loading = false;
  }
}
```

```typescript
private async handleAddSource(): Promise<void> {
  if (!this.tempSourceName || !this.tempSourceUrl) {
    Logger.warn('SettingsPage', 'Source name or URL is empty');
    return;
  }

  this.isAddingSource = true;

  try {
    // 验证URL格式
    try {
      new URL(this.tempSourceUrl);
    } catch (error) {
      Logger.error('SettingsPage', 'Invalid URL format');
      this.isAddingSource = false;
      return;
    }

    // 解析标签
    const tags = this.tempSourceTags
      ? this.tempSourceTags.split(',').map(tag => tag.trim()).filter(tag => tag)
      : [];

    // 创建新源
    const newSource: NetworkSourceConfig = {
      id: `source_${Date.now()}`,
      name: this.tempSourceName,
      url: this.tempSourceUrl,
      type: 'json',
      version: '1.0.0',
      isActive: true,
      priority: this.tempSourcePriority,
      tags: tags,
      contentUsage: {
        vodSites: true,
        liveChannels: true,
        wallpapers: true,
        adBlockRules: true,
        decoderConfigs: true
      }
    };

    await this.networkSourceManager.addSource(newSource);
    this.sources = this.networkSourceManager.getAllSources();
    await this.loadContentStats();

    Logger.info('SettingsPage', `Source added: ${newSource.name}`);
    this.closePopup();
  } catch (error) {
    const err = error instanceof Error ? error : new Error(String(error));
    Logger.error('SettingsPage', 'Failed to add source', err);
  } finally {
    this.isAddingSource = false;
  }
}
```
