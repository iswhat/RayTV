# RayTV 壳类型视频聚合器项目说明

## 项目来源

- **AndroidVer**：来自于 [https://github.com/FongMi/TV](https://github.com/FongMi/TV)
- **RayTV**：参考FongMi/TV，基于Harmony OS进行的全新开发实现

## 项目概述

RayTV是一个基于Harmony OS开发的专业壳类型视频聚合器，采用配置驱动的内容聚合架构。该应用不直接存储视频内容，而是通过动态配置源获取和整合来自多个第三方平台的视频资源，为用户提供统一的观看体验。

**重要声明：**
- 本项目仅用于Harmony OS程序开发学习目的
- 同时作为对Trae cn AI IDE能力的测试与验证
- 程序本身不具备也不集成任何站点的解析处理能力
- 项目通过配置源管理实现内容聚合，不直接处理视频内容
- 项目开发严格遵守相关法律法规，避免出现内容的版权问题

## 主要功能

作为学习用的壳类型视频聚合器，主要功能模块包括：
- 🎯 **配置驱动架构**：动态加载和管理多个视频内容源
- 🔄 **内容智能聚合**：统一解析和整合不同平台的视频资源
- 🧠 **智能推荐**：基于用户行为的个性化内容推荐，包括推荐算法和用户偏好分析（未使用AI大模型）
- ▶️ **增强播放体验**：自适应流媒体、画质优化、智能缓存、播放进度同步
- 🔍 **跨源统一搜索**：一次搜索遍历所有配置源，支持按源筛选和结果排序
- 🎛️ **插件化解析**：支持多种内容格式的可扩展解析器
- 🛡️ **智能广告拦截**：自动识别和过滤广告内容
- 👤 **用户行为分析**：记录用户行为，构建用户画像，提供个性化体验
- 📊 **实时性能监控**：系统性能指标收集和分析
- 🎨 **响应式UI设计**：可复用组件和现代化界面，包括7种风格的Button和Card组件
- 🤖 **Harmony OS集成**：系统特有功能完整演示
- 🏊 **内容池化**：统一的内容访问池，支持混合源架构
- 🏷️ **来源追踪**：清晰显示每个内容的来源
- 📊 **健康监控**：源健康状态实时监控
- 🔧 **站点管理**：完整的站点CRUD操作，支持导入导出和连接测试
- 📁 **NAS设备支持**：支持局域网内NAS或影音服务器的识别和访问，尤其支持华为家庭存储
- 🔍 **NAS目录浏览**：支持NAS设备中视频（影片）的目录浏览和索引建立
- 📱 **多设备管理**：支持多个NAS设备同时存在，当作多个“网络源”并列管理
- 🎬 **NAS视频播放**：支持直接播放NAS中的视频内容，无需下载

## 技术栈

- **Harmony OS**：应用开发框架
- **ArkTS**：主要开发语言（HarmonyOS特有）
- **TypeScript**：辅助开发语言
- **Ets**：Harmony OS UI组件
- **核心架构**：依赖注入 + 事件总线 + 服务层
- **现代化架构模式**：MVVM + 插件化架构
- **配置管理**：动态配置源加载机制
- **内容聚合**：多源内容整合引擎

## NAS功能使用说明

### 功能概述

RayTV支持局域网内NAS或影音服务器的识别和访问，尤其支持华为家庭存储。您可以通过以下方式使用NAS功能：

### 使用步骤

1. **NAS设备发现**
   - 进入"NAS管理"页面
   - 点击"扫描设备"按钮，系统会自动发现局域网内的NAS设备
   - 支持的设备类型：华为家庭存储、极空间ZSpace、小米Mi Home、群晖Synology、绿联UGreen、威联通QNAP等

2. **手动添加NAS设备**
   - 进入"NAS管理"页面
   - 点击"添加设备"按钮
   - 输入设备名称、IP地址、端口号
   - 选择设备类型和协议
   - 输入认证信息（如果需要）
   - 点击"保存"完成添加

3. **浏览NAS目录**
   - 在"NAS管理"页面选择已发现或添加的设备
   - 进入设备的目录浏览器
   - 浏览设备中的文件夹和文件
   - 支持按类型筛选、排序和搜索

4. **播放NAS视频**
   - 在目录浏览器中找到要播放的视频文件
   - 点击视频文件或播放按钮
   - 系统会跳转到播放页面开始播放
   - 支持播放控制、进度调整等功能

5. **管理NAS设备**
   - 在"NAS管理"页面可以查看所有已添加的设备
   - 支持设备的编辑、删除和状态查看
   - 可以设置设备的优先级和分组

### 支持的协议

- SMB/CIFS
- WebDAV
- HTTP
- 华为HiLink
- 极空间ZSpace
- 小米Mi Home
- 群晖Synology
- 绿联UGreen
- 威联通QNAP

### 注意事项

- 确保NAS设备和RayTV应用在同一局域网内
- 确保NAS设备已开启相应的服务和权限
- 对于需要认证的设备，请确保输入正确的用户名和密码
- 播放大文件时可能需要较长的加载时间，请耐心等待
