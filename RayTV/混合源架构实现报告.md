# RayTV混合源架构实现报告

## 项目概述

本次实现完成了RayTV应用的混合源架构,允许用户自由组合不同网络源的内容,实现更灵活的内容管理。

## 核心功能

### 1. 网络源独立管理
- 每个网络源作为独立的配置存储
- 支持源的添加、删除、编辑
- 支持源的启用/禁用切换
- 健康状态监控(健康/降级/不可达)

### 2. 内容池化架构
内容从各个源提取到统一的使用池:
- VodSitePool: 点播站点池
- LiveChannelPool: 直播频道池
- WallpaperPool: 壁纸池
- GlobalAdBlockRules: 全局广告过滤规则(合并所有源)
- GlobalDecoderConfigs: 全局解码配置(合并所有源)

### 3. 精细化内容控制
每个源可独立配置内容使用开关:
- vodSites: 是否使用点播站点
- liveChannels: 是否使用直播频道
- wallpapers: 是否使用壁纸
- adBlockRules: 是否使用广告过滤规则
- decoderConfigs: 是否使用解码配置
- parserConfigs: 是否使用解析器配置

### 4. 内容来源追踪
每个内容项标记来源信息:
- sourceId: 源ID
- sourceName: 源名称
- sourcePriority: 源优先级
- sourceTags: 源标签

### 服务层
1. **SourcePoolManager.ets** - 源池管理器
   - 位置: `raytv/src/main/ets/service/pool/SourcePoolManager.ets`
   - 功能: 管理统一内容池,从源加载内容

2. **ContentAccessService.ets** - 内容访问服务
   - 位置: `raytv/src/main/ets/service/content/ContentAccessService.ets`
   - 功能: 提供统一的内容访问接口

3. **NetworkSourceManager.ets** - 网络源管理
   - 位置: `raytv/src/main/ets/service/config/NetworkSourceManager.ets`

### UI组件
4. **NetworkSourceDetailPopup.ets** - 源详情弹窗
   - 位置: `raytv/src/main/ets/components/NetworkSourceDetailPopup.ets`
   - 功能: 显示和编辑源的详细信息

5. **ContentSourceBadge.ets** - 内容来源标识
   - 位置: `raytv/src/main/ets/components/ContentSourceBadge.ets`
   - 功能: 显示内容来源信息

     - MultiSourceBadge: 多源组合标识
     - SourceCard: 源卡片

6. **SettingsPage.ets** - 设置页面(完全重写)
   - 位置: `raytv/src/main/ets/pages/SettingsPage.ets`
   - 功能: 网络源管理UI、内容统计显示

## 优势总结

1. **灵活性**: 用户可以自由组合不同源的内容
2. **可维护性**: 源的添加/修改不影响其他源
3. **透明性**: 清晰显示每个内容的来源
4. **性能优化**: 池化访问,避免重复加载
5. **容错性**: 单个源故障不影响其他源的使用
6. **用户控制**: 用户可以精确控制每个源的内容使用

## 使用示例

### 添加网络源
```typescript
const source: NetworkSourceConfig = {
  id: 'source_custom_1',
  name: '自定义源',
  url: 'https://example.com/config.json',
  type: 'json',
  version: '1.0.0',
  isActive: true,
  contentUsage: {
    vodSites: true,
    liveChannels: true,
    wallpapers: false,
    adBlockRules: true,
    decoderConfigs: true,
    parserConfigs: false
  },
  priority: 3,
  tags: ['自定义', '推荐'],
  lastUpdated: Date.now(),
  healthStatus: 'healthy'
};

await networkSourceManager.addSource(source);
```

### 访问内容
```typescript
// 获取所有点播站点
const allSites = contentAccessService.getAllVodSites();

// 按源获取点播站点
const sitesFromSourceA = contentAccessService.getVodSitesBySource('source_a_id');

// 搜索点播站点
const searchResults = contentAccessService.searchVodSites('电影', {
  sourceIds: ['source_a_id', 'source_b_id'],
  categories: ['电影', '电视剧']
});

// 获取所有直播频道
const allChannels = contentAccessService.getAllLiveChannels();

// 获取随机壁纸
const wallpaper = contentAccessService.getRandomWallpaper();
```

### 搜索功能
```typescript
// 执行搜索
const searchRequest: SearchRequest = {
  query: '星际穿越',
  type: SearchType.VOD,
  sort: SearchSort.DEFAULT,
  page: 1,
  pageSize: 20,
  filters: {
    sourceIds: ['source_a_id'],
    minPriority: 2
  }
};

const result = await searchRepository.performSearch(searchRequest);

// 结果包含来源信息
result.items.forEach(item => {
  console.log(`${item.title} - 来源: ${item.sourceName}`);
});
```
