# HarmonyOS 6 应用开发学习计划

## 目标
**48 小时速成**专家级 HarmonyOS 6 应用开发专家，擅长 ArkTS + ArkUI 编程开发及问题解决

## 核心资源

### 官方资源
- **开发者官网**: https://developer.harmonyos.com/cn
- **DevEco Studio 下载**: https://developer.harmonyos.com/cn/develop/deveco-studio
- **官方文档**: https://developer.huawei.com/consumer/cn/doc/
- **社区论坛**: https://developer.harmonyos.com/cn/community

### 关键知识点

#### 1. ArkTS 语言基础
- TypeScript 超集，鸿蒙专用增强
- 声明式 UI 描述语法
- 状态管理与响应式编程
- 异步编程模型

#### 2. ArkUI 框架
- 声明式 UI 开发范式
- 基础组件（Text, Button, Column, Row 等）
- 布局系统（Flex, Grid, Stack 等）
- 动画与交互
- 自定义组件

#### 3. 核心能力
- 页面路由与导航
- 网络请求与数据处理
- 本地存储（Preferences, Database）
- 分布式能力
- 设备协同

#### 4. 高级主题
- ArkWeb（Web 与原生融合）
- 性能优化
- 调试与问题排查
- 上架与发布流程
